# Isolated system
Arcade game made with Unity where you control a spaceship avoiding asteroids and collecting points to refill fuel and increase your score.

## Features
- Inertia based movement
- Fast game loop
- 5 levels of difficulty with new types of asteroids
- Clean UI/UX 
- Online leaderboard
- Unlockable in-game skins

## Aditional features
- Original art
- Original music and SFX by WinningLess
